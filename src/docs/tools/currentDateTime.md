# Current Date/Time Tool

Returns the current date and time in both local and UTC formats.

## Overview

Use this tool whenever you need to know the current date, time, day of the week, or timezone. It returns both the local time (based on the system's timezone) and UTC time.

## Parameters

This tool takes no parameters.

## When to Use This Tool

- The user asks for the current date or time
- You need to generate timestamps or date-based content
- You need to know the day of the week or timezone

## Response Format

Returns a text block with:
- Local date and time with timezone
- UTC date and time
